Dashboard Home

Reusable starting point for tools, map, and documentation.

Welcome

This is the Version 2 dashboard home template. It is meant to be cloned into any environment (home lab, enterprise network, client site) and customized via config files instead of editing the HTML structure directly.

To get started, copy js/config.example.js to js/config.js and edit the values to match your environment.

Next Steps

  • Update your business, environment, org, and use-case in DASHBOARD_CONFIG.
  • Add or edit quick links pointing to your own tools.
  • Open tools/tools_home.html to define your tool list.
  • Open maps/maps_home.html to organize map pages and assets.

Live Widget Placeholder

This area can later be wired to:

  • Local JSON under core/data/
  • Simple scripts that export status to JSON
  • Backend APIs or webhooks returning JSON

The goal is to keep the HTML static while swapping in different data sources per environment.

Notes

All classes used here are defined in core/styles.css. Try to reuse those classes on other pages (tools, maps) so changes to the stylesheet automatically update the whole dashboard.
